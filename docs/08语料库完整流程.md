## 模块说明

### 1. 数据源采集 (pretrain/data-collection/)

预留模块，用于：
- GitHub代码仓库爬取
- Web数据采集
- 数据源管理

#### 1.1 GitHub代码采集
- **模块**: `pretrain/data-collection/`
- **功能**: 从GitHub爬取代码仓库，提取代码文本
- **输出格式**:
  ```json
  {
    "来源": "github",
    "仓库名": "author/repo",
    "文件名": "main.py",
    "ext": ".py",
    "path": "repo/src/main.py",
    "size": 1234,
    "原始编码": "utf-8",
    "md5": "abc123...",
    "text": "代码内容...",
    "时间": "20250318"
  }
  ```

#### 1.2 PDF文档采集
- **模块**: `pretrain/text-extraction/pdf_extractor.py`
- **功能**: 处理PDF文档，提取文本内容
- **输出格式**:
  ```json
  {
    "section": "章节标题",
    "content": "章节内容..."
  }
  ```

### 2. 文本提取 (text-extraction/)

#### 2.1 PDF提取
- **工具**: MinerU, PyMuPDF, pdfplumber
- **支持**: OCR识别、表格提取、图片处理
- **输出**: Markdown格式

#### 2.2 代码提取
- **工具**: charset_mnbvc (编码检测)
- **支持**: 多种编程语言、多种编码格式

### 3. 数据清洗 (data-cleaning/)

#### 3.1 文本清洗
- 移除图片、链接、目录、乱码
- 移除参考文献、页眉页脚
- 修复编码问题

#### 3.2 语义去重
- 使用Sentence-BERT进行语义相似度计算
- 可配置相似度阈值（默认0.9）

### 4. 格式转换 (format-conversion/)

#### 4.1 TRL训练格式（新增）

**SFT格式 - Conversational**:
```json
{
  "messages": [
    {"role": "user", "content": "用户问题"},
    {"role": "assistant", "content": "助手回答"}
  ]
}
```

**SFT格式 - Text**:
```json
{
  "text": "完整的文本内容..."
}
```

**SFT格式 - Prompt-Completion**:
```json
{
  "prompt": "问题描述或上下文",
  "completion": "期望的输出"
}
```

**DPO格式**:
```json
{
  "prompt": "用户问题",
  "chosen": "优选回答",
  "rejected": "拒绝回答"
}
```

### 5. 数据验证 (sft/validation/)

- **模块**: `sft/validation/validate_dataset.py`
- **功能**: 格式验证、质量评估

#### 5.1 格式验证
- JSON格式检查
- 必需字段检查
- 数据类型验证

#### 5.2 质量评估
- 文本长度统计
- 语言分布
- 质量分数

### 6. 数据集划分 (tools/dataset/)

- **模块**: `tools/dataset/dataset_utils.py`
- **功能**: 数据集划分、合并、采样、分析

```bash
# 划分数据集
python -m tools.dataset.dataset_utils split -i ./data.jsonl -o ./dataset --test-size 0.1

# 合并数据集
python -m tools.dataset.dataset_utils merge -i ./data1 ./data2 -o ./merged.jsonl

# 采样
python -m tools.dataset.dataset_utils sample -i ./large.jsonl -o ./sampled.jsonl --ratio 0.1

# 分析
python -m tools.dataset.dataset_utils analyze -i ./data.jsonl
```

### 7. QA构建 (sft/qa-construction/)

预留模块，用于：
- 从文本自动构建问答对
- 多轮对话构建
- 指令数据生成

### 8. 通用工具 (tools/utils/)

预留模块，用于：
- 文本处理工具
- 编码转换工具
- 其他通用功能

## 完整流程示例

### 示例1: 代码语料处理

```bash
# 1. 准备仓库列表
echo "12345,https://github.com/user/repo.git" > repos_list.txt

# 2. 使用预训练流水线处理
python scripts/run_pretrain_pipeline.py --config configs/pretrain.yaml

# 3. 或分步执行 - PDF提取
python -m pretrain.text-extraction.pdf_extractor -i ./pdfs -o ./output/extracted

# 4. 数据清洗
python -m pretrain.cleaning.text_cleaner -i ./output/extracted -o ./output/cleaned

# 5. 去重
python -m pretrain.deduplication.semantic_dedup -i ./output/cleaned/merged.jsonl -o ./output/deduped.jsonl -m hybrid

# 6. 格式转换
python -m pretrain.format.convert_to_pretrain -i ./output/deduped.jsonl -o ./output/pretrain.jsonl
```

### 示例2: PDF文档处理

```bash
# 1. 提取PDF内容
python -m pretrain.text-extraction.pdf_extractor -i ./pdfs -o ./output/extracted --method mineru

# 2. 数据清洗
python -m pretrain.cleaning.text_cleaner -i ./output/extracted -o ./output/cleaned --type markdown

# 3. 语义去重
python -m pretrain.deduplication.semantic_dedup -i ./output/cleaned/merged.jsonl -o ./output/deduped.jsonl -m hybrid

# 4. 格式转换
python -m pretrain.format.convert_to_pretrain -i ./output/deduped.jsonl -o ./output/pretrain.jsonl

# 5. 划分数据集
python -m tools.dataset.dataset_utils split -i ./output/pretrain.jsonl -o ./output/final/ --test-size 0.1
```

### 示例3: SFT语料生成

```bash
# 1. 设置API密钥
export DEEPSEEK_API_KEY="your-api-key"

# 2. 生成问答对
python -m sft.data-generation.generate_qa --api deepseek -i ./contexts.txt -o ./qa.jsonl --num-questions 3

# 3. 格式转换
python -m sft.format-conversion.convert_to_sft -i ./qa.jsonl -o ./sft.jsonl

# 4. 数据验证
python -m sft.validation.validate_dataset -i ./sft.jsonl

# 5. 划分数据集
python -m tools.dataset.dataset_utils split -i ./sft.jsonl -o ./dataset --test-size 0.1 --val-size 0.1
```

## 与HuggingFace训练的集成

### 使用TRL进行训练

```python
from datasets import load_dataset
from trl import SFTTrainer, SFTConfig
from peft import LoraConfig

# 加载处理好的数据集
dataset = load_dataset("json", data_files="output/sft/train.jsonl")

# 创建训练器
trainer = SFTTrainer(
    model="Qwen/Qwen2.5-7B",
    train_dataset=dataset["train"],
    args=SFTConfig(
        output_dir="output/model",
        num_train_epochs=3,
        per_device_train_batch_size=4,
        learning_rate=2e-5,
        push_to_hub=True,
        hub_model_id="username/your-model",
    ),
    peft_config=LoraConfig(r=16, lora_alpha=32),
)

# 开始训练
trainer.train()
```

## 常见问题

### Q1: 如何处理多语言数据？
建议按语言分开处理，分别训练或使用多语言基座模型。

### Q2: 数据量需要多大？
- 微调：通常需要1K-10K高质量样本
- 预训练：需要更大规模数据（百万级以上）

### Q3: 如何评估数据质量？
使用提供的验证工具，检查格式、长度分布、语言分布等指标。

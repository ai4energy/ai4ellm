## 基座模型选型（已定死）

- **主力**：Qwen3.6（中文 + 领域适配，不回退）
- **补充**：gemma-4-31B（多语言 + 开源生态）

- **首发**：ModelScope（魔搭），国内能源领域占坑
- **同步**：HuggingFace，全球影响力

## 蒸馏方案

教师模型（通义千问 API / GPT-4o）→ 生成高质量 SFT 数据 → 蒸馏到 Qwen3.6-35B-A3B（学生）

## 技术架构

PSE（过程系统工程）的扩大化：
- 材料（微观/介观）→ 物性计算、分子设计
- 化工（单元/流程）→ 反应工程、分离过程
- 能源（系统/网络）→ 能量平衡、热电耦合
- 电气（设备/电网）→ 电力系统、电机控制

共性：建模 → 仿真 → 优化 → 控制

## 内容引流 + 变现

- 能源数字化：专业赛道，高客单价
- 中医：大众赛道，资料+社群轻量闭环
- 两条腿走路，互补

## 容器化部署

- **海光 DCU**：`Dockerfile.rocm`，基于 `rocm/dev-ubuntu-22.04:6.0`
- **NVIDIA GPU**：`Dockerfile.cuda`，基于 `nvidia/cuda:12.1-...`
- **K8s 部署**：曙光用 K8s 调度，挂载存储卷 + Secret 注入 API 密钥
- **语料处理不依赖 GPU** — 主要是文本提取、清洗、去重、API 调用
- **模型微调才需要 GPU** — 训练阶段用 pytorch-rocm 版本
